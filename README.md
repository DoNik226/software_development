# Dungeon Delver
Автор: Доскал Никита

"Dungeon Delver" — это небольшая рогалик-игра в жанре dungeon crawler, где игрок берет на себя роль искателя приключений, исследующего случайно сгенерированные подземелья, наполненные врагами, ловушками и сокровищами. Каждый этап предлагает уникальные задачи и возможности, основанные на различных алгоритмах, что делает каждую игровую сессию уникальной.
