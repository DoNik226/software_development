# Dungeon Delver
Автор: Доскал Никита

"Dungeon Delver" — это небольшая рогалик-игра в жанре dungeon crawler, где игрок берет на себя роль искателя приключений, исследующего случайно сгенерированные подземелья, наполненные врагами, ловушками и сокровищами. Каждый этап предлагает уникальные задачи и возможности, основанные на различных алгоритмах, что делает каждую игровую сессию уникальной.

1. **Пользователь может начать новое приключение**:
   Пользователь нажимает кнопку "Начать игру". Приложение отображает случайно сгенерированное подземелье и вводит игрока в игру.

2. **Пользователь может перемещать персонажа**:
   Пользователь нажимает клавиши-стрелки для перемещения. Приложение обновляет положение персонажа на экране и отображает новую карту уровня.

3. **Пользователь может взаимодействовать с объектами**:
   Пользователь нажимает клавишу "E" вблизи объекта. Приложение проверяет, является ли объект взаимодействуемым, и отображает соответствующее сообщение или выполняет действие (например, подбирает предмет).

4. **Пользователь может сражаться с врагами**:
   Пользователь нажимает кнопку "Атака". Приложение рассчитывает вероятность удара, использует алгоритм броска кубика и отображает результат (урон, пропуск удара).

5. **Пользователь может получать квесты**:
   Пользователь взаимодействует с NPC. Приложение отображает доступные квесты с их описаниями и наградами.

6. **Пользователь может использовать навыки**:
   Пользователь нажимает клавишу для активации навыка. Приложение проверяет, доступен ли навык, и отображает его эффект (например, лечение или урон).

7. **Пользователь может осматривать инвентарь**:
   Пользователь нажимает кнопку "Инвентарь". Приложение отображает список предметов, собранных игроком, с возможностью использования или выбрасывания.

8. **Пользователь может видеть текущее здоровье**:
   При получении урона приложение обновляет индикатор здоровья на экране и отображает сообщение о текущем состоянии (например, "Здоровье: 3/10").

9. **Пользователь может завершить игру**:
    Пользователь нажимает кнопку "Выйти". Приложение сохраняет прогресс и закрывает игру.
